## 🎯 Sobre o Projeto

**Poker Hands Manager** - Sistema profissional completo para gerenciamento automatizado de históricos de mãos de poker.

- **Localização:** `C:\Users\paulo\OneDrive\Desktop\poker-hands-manager`
- **Objetivo:** Automatizar 100% do processo de solicitação, recebimento e processamento de históricos de mãos via Discord
- **Economia de tempo:** 80-90% (de 30-60 min manual para 5 min automatizado)

## 🔜 Próximos Passos

### 1. Configurar Bot Discord
- Criar aplicação no Discord Developer Portal
- Obter Token do bot
- Obter Guild ID (Server ID)
- Criar arquivo `bot/.env`
- Instalar dependências: `cd bot && npm install`

### 2. Configurar Dashboard
- Criar arquivo `dashboard/.env`
- Instalar dependências: `cd dashboard && npm install`

### 3. Testar Sistema Completo
- Iniciar backend: `cd backend && npm start`
- Iniciar bot: `cd bot && npm start`
- Iniciar dashboard: `cd dashboard && npm run dev`
- Ou usar: `start-all.bat` (Windows)

### 4. Testes Funcionais
- Registrar jogador
- Solicitar mãos
- Enviar arquivo de teste
- Verificar no dashboard
